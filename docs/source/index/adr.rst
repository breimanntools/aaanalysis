Architecture Decisions
======================
Architecture Decision Records (ADRs) document non-obvious design decisions for
AAanalysis. Each entry covers the choice that was made, the alternatives that
were considered, and the downstream consequences — providing context for
readers and future maintainers about *why* the code is shaped the way it is.
