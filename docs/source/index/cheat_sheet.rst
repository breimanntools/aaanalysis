Cheat Sheet
===========

A one-glance reference to the canonical AAanalysis workflow, the main classes
grouped by capability, the prediction levels (residue / domain / protein), and
the feature ontology (*Part × Split × Scale*). Every code snippet uses the
public API and the canonical vocabulary.
